Need help finding these?
  1. Install the Huckleberry HA integration and sign in.
  2. In HA: Profile → Security → Long-Lived Access Tokens → Create Token. Copy it into the token field above.
  3. In HA: Settings → Devices & Services → Huckleberry. Click your child's device — the device ID is shown at the top.
